## Versões Futuras

### v0.1.0 - Fundação (Previsto: Semana 2)
- Shell visual funcional
- Temas Dark/White implementados
- Navegação lateral básica

### v0.2.0 - Editor (Previsto: Semana 4)
- Monaco Editor integrado
- Syntax highlighting para XML
- Sistema de abas múltiplas

- Envio de requisições SOAP
- Captura e exibição de respostas

- Salvamento local em JSON
- Importação/Exportação de coleções
- Histórico de requisições

### v1.0.0 - Lançamento (Previsto: Semana 15)
- Performance otimizada (< 150MB RAM)
- Instaladores para Windows, macOS e Linux
