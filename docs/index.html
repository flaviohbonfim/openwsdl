Por que OpenWsdl?

🚀

Alta Performance

Menos de 150MB de RAM em repouso e inicialização em menos de 1.5 segundos.

⚡

Editor Monaco

O mesmo motor do VS Code para edição de XML com syntax highlighting e atalhos globais.

🔍

Parser Robusto

Suporte completo a WSDL 1.1/1.2 com recursão de XSD e geração de envelopes.

📦

Cross-Platform

Binários nativos para Windows, macOS e Linux sem a lentidão do Java.

Como Instalar

🖥️ Desktop

Acesse a nossa página de Releases e baixe o instalador para o seu sistema.

Apple macOS

Como o projeto é open-source, você precisará autorizar a abertura:

  1. Clique com o botão direito no app e selecione "Abrir".
  2. No aviso, clique em "Abrir" novamente.

🛠️ Desenvolvedores

git clone https://github.com/flaviohbonfim/openwsdl.git
cd openwsdl
flutter pub get
flutter run -d windows
